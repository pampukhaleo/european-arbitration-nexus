## Что строим

Отдельная одноязычная (украинская) посадочная страница про **МКАС при ЕАП** и арбитражную оговорку. Контент — строго из ТЗ. Главное действие — кнопки прямого контакта (звонок / email / WhatsApp). На основной сайт (en/fr/ru) изменения минимальные — просто добавляется новый маршрут.

## Маршрут и язык

- URL: `/landing` (плюс `/?/landing` через spa-github-pages, но тебе для рекламы хватит просто `/landing`).
- Один язык — украинский. Без переключателя языков, без изменения общего `LanguageContext`.
- В `<html lang="uk">` через `<Seo lang="uk" />` (не ломает остальные страницы — Helmet перезаписывает на каждой).

## Технические детали

**Файлы:**
- `src/pages/Landing.tsx` — страница (UA-тексты прямо в JSX, без LanguageContext, чтобы не трогать общий i18n)
- `src/components/landing/LandingHeader.tsx` — минимальная шапка только с лого (`<Link to="/">`)
- `src/components/landing/LandingFooter.tsx` — минимальный футер
- `src/components/landing/StickyContactBar.tsx` — плавающая мобильная CTA-панель
- `src/App.tsx` — добавить `<Route path="/landing" element={<Landing />} />` (без Layout, чтобы не тащить общую шапку/футер сайта с меню)

**Стиль:** существующая палитра (`eac-primary`, `eac-dark`, `eac-light`), Tailwind, shadcn `<Button>`. Никаких новых зависимостей.

**SEO/индексация:**
- `<Seo lang="uk" robots="noindex, nofollow" ... />` — рекламный лендинг не должен индексироваться (не конкурирует с основным сайтом, не путает Google).
- og:title/og:description под FB-превью.
- Подготовлю «слоты» для будущих пикселей (Meta Pixel, GA event на клик по CTA) в виде комментариев `// TODO: pixel` — реально подключим, когда дашь Pixel ID.

**Ссылка из ТЗ** на `/arbitration/clause` (она уже существует в роутере) — добавлю как кнопку «Завантажити шаблон застереження».

## Что НЕ делаем

- Не добавляем форму (по твоему выбору — только кнопки контакта).
- Не добавляем украинский в общий `LanguageContext` (только эта страница на UA).
- Не трогаем основной сайт, шапку, футер, стили.
- Без формы → без новых таблиц БД, без edge-функций, без email-уведомлений.

## Проверка после реализации

- `/landing` открывается, шрифты/цвета как на сайте.
- Клик по «Подзвонити» открывает диалер; WhatsApp — вкладку wa.me; Email — почтовый клиент.
- На мобильном видна нижняя плавающая панель.
- На desktop её нет.
- В исходнике страницы стоит `<meta name="robots" content="noindex,nofollow">`.
- Клик по логотипу ведёт на главную сайта (`/`).
